# README.md

## Телеграм Бот MessBo (@messbobot)

MessBo - Доска Объявлений это Telegram-бот, позволяющий создавать личные и общие доски объявлений в чатах. Проект написан студентом Бердниковым А.С. группы ДИС-233/21.Б на Python в рамках учебной программы.

---

### Планируемый функционал

- Просмотр всех своих, просмотр одной, редактирование и удаление досок
- Добавление белого списка пользователей, которым разрешается модерировать доску
- Добавление записей на доску
- Добавить вывод отказа в действии
- Добавить кнопки

### Как запустить проект

1. Клонировать репозиторий и зайти в него:
```bash
git clone https://github.com/Tapotheslipper/messbo.git
cd messbo
```
2. Создать виртуальное окружение, запустить его и установить зависимости:
```bash
python -m venv venv

source venv/bin/activate (Linux/MacOS)
```
```bat
venv/Scripts/activate (Windows)
```
```bash
pip install -r requirements.txt
```
3. Скопировать файл .env.example в .env и вставить необходимый токен
4. Запустить:
```bash
python main.py
```

---

### Команды бота

* /help
    Выводит список команд
* /new_board <название>
    Создаёт новую доску, привязанную к пользователю, как владельцу. Без указанного названия оно генерируется автоматически

### Файловая структура

```
/bot/
    /board_manager.py      # логика создания и управления досками
    /create_bot.py         # создание бота
    /handlers.py           # обработка сообщений в чате
    /run_bot.py            # запуск бота
    /utils.py              # вспомогательные функции
/config/
    /config.py             # конфигурация
/database/
    /connection.py         # подключение к базе данных
/models/
    /Board.py              # модель доски
    /BoardEntry.py         # модель записи на доске
    /Logger.py             # модель логгера
    /User.py               # модель пользователя
/.env.example              # пример необходимого окружения
/CHANGELOG.md              # список основных изменений по версиям
/main.py                   # точка входа в бота
/requirements.txt          # используемые зависимости
```

---

### Основные используемые зависимости

* pyTelegramBotApi -> Библиотека для создания Telegram ботов.
* sqlite3          -> Встроенная библиотека Python 3.x для использования SQLite.
* python-dotenv    -> Возможность доставать значения из переменных окружения .env.
* requests         -> Библиотека для отправки HTTP запросов.
